# Fantateam

Fantateam è un progetto sperimentale nato per aiutare gli appassionati di Fantacalcio a prendere decisioni più informate. L'obiettivo è creare una piattaforma semplice che possa evolversi nel tempo, integrando strumenti utili, analisi e – in futuro – anche componenti di intelligenza artificiale.

## 🚀 Cosa puoi fare

- Esplorare l’interfaccia semplice della demo
- Leggere descrizioni di funzionalità future
- Restare aggiornato sullo sviluppo

## 🛠️ In sviluppo

Attualmente Fantateam è in una fase di ridefinizione. Alcune funzionalità saranno aggiunte nei prossimi mesi in base a disponibilità e feedback.
